AI Trading Radar - PANDUAN SERIAL KEY UNTUK PEMBUAT APLIKASI
===========================================================

Ringkasan
---------
Aplikasi sekarang wajib punya SERIAL KEY NUMBER sebelum bisa jalan.
Serial key disimpan di:

  %APPDATA%\AITradingRadar\license.json

Jika serial belum ada, salah, atau expired, aplikasi akan meminta serial baru.


Cara generate serial key
------------------------
Jalankan dari folder project ini:

  python tools\make_serial.py --plan 1m --customer "Nama Customer"

Pilihan plan:

  1m        = 1 bulan
  3m        = 3 bulan
  6m        = 6 bulan
  1y        = 1 tahun
  lifetime  = tanpa expired

Contoh:

  python tools\make_serial.py --plan 1m --customer "Customer A"
  python tools\make_serial.py --plan 3m --customer "Customer B"
  python tools\make_serial.py --plan 6m --customer "Customer C"
  python tools\make_serial.py --plan 1y --customer "Customer D"
  python tools\make_serial.py --plan lifetime --customer "Customer E"


Tanggal expired custom
----------------------
Kalau ingin menentukan tanggal expired manual:

  python tools\make_serial.py --plan 1y --expires 2027-12-31 --customer "Customer A"


Aktivasi di laptop customer
---------------------------
Saat aplikasi pertama kali dibuka, masukkan serial yang dihasilkan tool.
Serial juga bisa diaktivasi lewat command:

  AITradingRadar.exe --license-key "AIB-XXXXX-XXXXX-XXXXX-XXXXX-XXXXX-XXXXX-X"

Cek status lisensi:

  AITradingRadar.exe --license-info

Reset lisensi tersimpan:

  AITradingRadar.exe --reset-license


Catatan keamanan
----------------
Jangan ikutkan folder tools atau source code ke customer. Kirim hanya installer
atau portable ZIP yang ada di folder release.

Sistem ini offline, jadi cukup untuk distribusi sederhana. Untuk proteksi kuat
terhadap cracking, perlu aktivasi online/server lisensi.
